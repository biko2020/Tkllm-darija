Integration with Darija NLP Ecosystem (May 2026 Update)
🔗

Leverage Existing Resources: The Darija NLP landscape has matured significantly in 2024-2025, with major releases in early 2025. Key datasets and models to integrate:

  • → Atlas-Chat (MBZUAI & AtlasIA, Sept 2024): 458K instruction dataset, Gemma 2-based models (2B/9B/27B), DarijaMMLU/HellaSwag benchmarks. First modern LLM specifically for Moroccan Darija.
  • → GemMaroc (May 2025 - Latest): Green AI approach achieving 61.6% DarijaMMLU and 60.5% HellaSwag with minimal compute (48 GPU·h). Quality-over-quantity LoRA tuning on ~5K instructions. Bilingual reasoning preserved.
  • → TerjamaBench (Jan 2025): Cultural benchmark for English-Darija MT with meticulously curated parallel texts in Arabic-script Darija and Arabizi. Shows limitations of standard MT metrics.
  • → Terjman v2 (AtlasIA, March 2025): Updated translation models (Nano/Small/Base/Large) with improved efficiency and accuracy on larger, refined datasets.
  • → DODa v2 (May 2024): Darija Open Dataset expanded to 150K+ entries with dual-script support (Arabic + Latin). Includes 86K+ translated sentences.
  • → DVoice: Community ASR dataset for African languages. Use SpeechBrain's wav2vec2-dvoice-darija as baseline model.
  • → DarijaBanking: Domain-specific dataset for banking intent classification, useful for future vertical expansion.

Implementation Strategy: Use these resources to bootstrap ML models (start with DVoice + Whisper for ASR, Atlas-Chat benchmarks for quality), validate quality scores, and benchmark platform performance against state-of-the-art. Prioritize GemMaroc's green AI approach for cost-effective fine-tuning.
